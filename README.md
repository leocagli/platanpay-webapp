# 🍌 PlatanPay WebApp

**Tu agente IA que encuentra las mejores ofertas y paga por vos.**

Demo interactiva completa de la WebApp de **PlatanPay** — una plataforma de IA para compras inteligentes en Latinoamérica con integración a Mercado Pago.

![PlatanPay WebApp Demo](https://raw.githubusercontent.com/leocagli/platanpay-webapp/main/index.html)

## ✨ Características

- **Login / Demo rápido** con estado persistente
- **Dashboard** con métricas en tiempo real (saldo, presupuesto, ahorro mensual)
- **Ofertas del Agente IA** — 4 propuestas exclusivas con selección múltiple
- **Flujo completo de Human-in-the-Loop**: seleccionar → revisar → aprobar y pagar
- **Historial de ahorros** dinámico (se actualiza después de cada compra)
- **Chat con Agente IA** (simulado pero con respuestas contextuales)
- **Perfil** con datos del usuario
- **Totalmente responsive** — funciona perfecto en desktop, tablet y móvil
- **Actualizaciones en vivo** del saldo y presupuesto después de cada compra
- **Confeti + animaciones** en compras exitosas
- **Self-contained** — un solo archivo HTML (Tailwind CDN + FontAwesome)

## 🚀 Cómo usar

1. Abrí `index.html` en cualquier navegador moderno
2. Entrá con **modo Demo** (botón inferior) o con cualquier email + contraseña
3. Explorá el Dashboard, seleccioná ofertas, revisá y aprobá compras
4. Mirá cómo se actualiza tu saldo y el historial en tiempo real
5. Probá el chat con el Agente IA

## 🛠️ Stack

- Vanilla HTML + Tailwind CSS (CDN)
- JavaScript puro con state management + localStorage
- Diseño mobile-first pero optimizado para escritorio
- Totalmente funcional sin backend (demo con datos simulados)

## 📦 Despliegue

Listo para GitHub Pages:
- Subí la carpeta o el `index.html` directamente
- Activá GitHub Pages en la rama `main`

## 📝 Sobre el proyecto

PlatanPay es una propuesta de agente IA multi-agente para compras en LatAm (Argentina, Chile, etc.) que:
- Busca ofertas personalizadas en Carrefour, Mercado Libre, Club Bepo, etc.
- Muestra ahorro real en cada producto
- Requiere aprobación humana antes de pagar
- Integra pago con Mercado Pago
- Lleva control de presupuesto y ahorro mensual

**Versión actual:** WebApp v1.6 (responsive desktop + mobile)
